## 已知问题和解决方案

### 问题1: 首次启动需要安装依赖
**解决方案**: 
```bash
pip install -r requirements.txt
```
已在文档中明确说明

### 问题2: 需要创建.env文件
**解决方案**: 
```bash
copy .env.example .env
```

### 问题3: 数据库需要手动初始化
**解决方案**:

mysql -h 127.0.0.1 -P 3310 -u root -p < init.sql

已在文档中提供多种方法
